API keys are a way to grant third-party tools and applications partial read-only access to your Guild Wars 2 account, without revealing any login information like user or password.

In order to create an API key you should login to your account on the GuildWars2 official page and go to the API Keys section in your account options.

Below is a convenient link there. Feel free to manually navigate there, it's never too paranoid to be suspicious of unkown sites that give you direct links to login screens!

https://account.arena.net/applications
